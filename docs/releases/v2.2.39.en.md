# Twitch Drop Farmer v2.2.39

This release focuses on UI polish, translations, and a small onboarding help note.

## Added

- A discreet notice below the dashboard controls explaining that a just-selected game may briefly show "0/0 min" or "No active drops", and that clicking "Refresh dashboard" forces a full whitelist scan. The "Refresh dashboard" button also gained a tooltip describing what it does and that it can take a few minutes.

## Changed

- Full review of the PT-PT (pre-1990 orthographic agreement) and EN translations. Every UI string and user-visible log message now uses the old PT-PT spelling consistently (`actual`, `activo`, `selecção`, `detectar`, `actualizar`), with accents fixed on several notices/tooltips that were previously unaccented. The English strings were reviewed and made consistent (`behaviour`).

## Fixed

- The "About" dialog showed `\n-` as literal text because the body used escaped `\\n` instead of real line breaks. It now renders proper line breaks, and the wording was rewritten to reflect current features (streamless mode, sticky per-game targeting, full dashboard scan, filters/diagnostics/JSON session) in both languages.

## Maintenance

- Pre-release security check: confirmed no private data (token, cookies, login, email, user IDs, or local paths) is present in the repository files.

## Result

Version 2.2.39 makes the interface clearer and linguistically consistent, and fixes the malformed About dialog.
