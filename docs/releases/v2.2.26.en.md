# Twitch Drop Farmer v2.2.26

This release fixes a classification issue that could mark normal campaigns as subscription-required.

## Highlights

- Fixed false-positive "subscription required" labels in the dashboard.
- Browser fallback text heuristics are now stricter and safer.
- Fresh zip artifact generated for version 2.2.26.

## Fixes

- Subscription-only detection is now more conservative to avoid misclassifying campaigns that still have watchable drops.
- Removed broad subscription term matches in fallback parsing and kept only explicit phrase checks.

## Result

Version 2.2.26 improves campaign listing reliability and correct drop visibility.
