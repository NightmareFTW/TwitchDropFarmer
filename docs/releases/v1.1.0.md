# Twitch Drop Farmer v1.1.0

This release focuses on reliability, smoother UI behaviour, and much better game artwork loading.

## Highlights

- Added asynchronous game artwork loading to keep the dashboard and farming view responsive.
- Improved box art resolution with a prioritized fallback chain: Twitch data first, then Steam, then Google Images.
- Added a generated visual placeholder with game initials when no real artwork can be resolved.
- Added a manual dashboard refresh action to force artwork and snapshot refreshes.

## Fixes

- Reduced long GraphQL stalls by enforcing a total retry timeout.
- Hardened timestamp parsing against malformed Twitch data.
- Fixed browser fallback cleanup to avoid leaking Qt web resources.
- Improved campaign completion detection for campaigns that stay active after their rewards are already finished.
- Stopped caching failed artwork lookups so missing images can be retried later.
- Verified Steam library artwork URLs before using them.

## Result

The app should now feel more fluid during refreshes, recover better from temporary Twitch/API failures, and show far fewer missing game images.
