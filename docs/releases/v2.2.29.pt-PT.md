# Twitch Drop Farmer v2.2.29

Esta versao corrige o painel "Drops activos" vazio para campanhas que a Twitch so expoe como dados de resumo.

## Destaques

- Corrigido o painel de drops activos a mostrar "Sem drops ativos para mostrar" mesmo com uma campanha a ser activamente farmada.
- Causa raiz: o integrity check da Twitch bloqueia a query de detalhe por-campanha (`DropCampaignDetails`), e o fallback de interceção de GQL do browser so visita a pagina de listagem, nao a vista de detalhe de cada campanha, pelo que `timeBasedDrops` fica muitas vezes em falta.

## Correcao de problemas

- `_refresh_active_drops_panel` passa agora a usar o progresso agregado da campanha (`next_drop_name`, `required_minutes`, `progress_minutes` — campos que a Twitch devolve mesmo sem detalhe por-drop) como item unico quando nao ha lista de drops discriminada.

## Resultado

A versao 2.2.29 faz o painel de drops activos refletir o progresso real de farming mesmo quando a Twitch nao fornece detalhe por-drop.
