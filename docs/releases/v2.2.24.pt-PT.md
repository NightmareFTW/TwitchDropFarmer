# Twitch Drop Farmer v2.2.24

Esta versao reforca a reconciliacao do modo streamless e fecha pequenos pontos de manutencao da release, com foco em progresso real e higiene de empacotamento.

## Destaques

- Reconciliação do progresso streamless mais robusta quando a Twitch atrasa o inventário.
- Sinal complementar via `CurrentDrop` para reduzir falsos estados de "sem avanço".
- Mapeamento interno de drops melhorado para diagnóstico mais fiável.
- Limpeza de artefactos de release gerados localmente.

## Correcao de problemas

- Evitado o congelamento aparente do progresso quando `sendSpadeEvents` é aceite mas o Inventory ainda não reflecte os minutos mais recentes.
- Corrigidos escapes de regex JS embebida para eliminar warnings de runtime no arranque/testes.

## Resultado

A versao 2.2.24 fica alinhada com o codigo actual e com as notas de changelog desta release.
