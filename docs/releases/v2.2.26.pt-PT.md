# Twitch Drop Farmer v2.2.26

Esta versao corrige um problema de classificacao que marcava campanhas normais como se exigissem subscricao.

## Destaques

- Correcao de falso positivo "exige subscricao para resgatar" no dashboard.
- Heuristicas de texto do fallback do browser mais restritas e seguras.
- Novo artefacto zip gerado para a versao 2.2.26.

## Correcao de problemas

- A deteccao de campanhas sub-only foi tornada mais conservadora, evitando classificacoes incorrectas de campanhas com drops watchable.
- Removidos matches genericos por termos de subscricao no parsing de fallback; ficam apenas frases explicitas.

## Resultado

A versao 2.2.26 melhora a fiabilidade da listagem de campanhas e a visibilidade correcta dos drops.
