# Twitch Drop Farmer v2.2.27

This release definitively fixes incorrect subscription-only campaign classification.

## Highlights

- Fixed global false-positive "subscription required" labeling in the dashboard.
- More conservative sub-only inference for incomplete drop payloads.
- New regression tests added to prevent recurrence.

## Fixes

- Campaigns with drops missing `requiredMinutesWatched` are no longer auto-classified as subscription-only.
- Mixed campaigns with watchable drops are no longer blocked by isolated subscription metadata.
- Sub-only is now flagged only on explicit structural signals or when all known drop requirements are zero.

## Result

Version 2.2.27 improves campaign detection and restores correct drop visibility in the dashboard.
