# Twitch Drop Farmer v2.2.31

Esta versao estende o fallback invisivel de browser para obter detalhe real por-drop das campanhas.

## Destaques

- O proprio sistema de integrity check da Twitch tem tido falhas intermitentes e prolongadas para pedidos GraphQL directos (confirmado externamente pelo mantenedor do DevilXD/TwitchDropsMiner, que teve o mesmo erro no seu proprio browser real). O fallback invisivel (headless, sem janela) continua a ser a forma fiavel de alcancar a Twitch durante esses periodos.
- Esse fallback so visitava antes a pagina de listagem de campanhas, que devolve dados-resumo sem `timeBasedDrops` — deixando o painel de drops activos e o progresso da campanha vazios para muitas campanhas activas.

## Adicionado

- `_campaigns_via_browser_gql_intercept` passa agora a visitar tambem a pagina de detalhe de cada campanha activa (`/drops/campaigns?dropID=<id>`) na mesma pagina invisivel, capturando o `DropCampaignDetails` real que o proprio JavaScript da Twitch dispara. Limitado a 40 campanhas por ciclo, priorizando as que terminam mais cedo. Nunca abre nenhuma janela de browser.

## Resultado

A versao 2.2.31 deve preencher o progresso real dos drops mesmo quando o integrity check da API directa da Twitch nao esta disponivel, ao custo de um ciclo de actualizacao mais lento quando este caminho e usado — uma troca explicitamente aceite pelo utilizador em favor de nunca abrir uma janela de browser.
