# Twitch Drop Farmer v2.2.29

This release fixes an empty "Active drops" panel for campaigns Twitch only exposes as summary data.

## Highlights

- Fixed the active-drops panel showing "No active drops to display" even while a campaign is being actively farmed.
- Root cause: Twitch's integrity check blocks the per-campaign detail query (`DropCampaignDetails`), and the browser GQL-intercept fallback only visits the campaign list page, not each campaign's detail view, so `timeBasedDrops` is often missing.

## Fixes

- `_refresh_active_drops_panel` now falls back to the campaign-level aggregate progress (`next_drop_name`, `required_minutes`, `progress_minutes` — fields Twitch returns even without per-drop detail) as a single summary item when no itemized drop list is available.

## Result

Version 2.2.29 makes the active-drops panel reflect real farming progress even when Twitch withholds per-drop detail.
