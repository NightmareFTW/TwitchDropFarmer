# Twitch Drop Farmer v2.2.38

This release adds visibility into how many whitelisted games are actually showing up on the dashboard.

## Added

- A new label next to the "Hide games with subscription-only drops" checkbox shows "Showing X/Y whitelisted games" (or "Showing X game(s)." when no whitelist is configured and every available game is shown). `Y` is the number of whitelist entries configured; `X` is how many actually render as cards -- the gap covers both games with no known campaign yet (name not matched in Twitch's data) and games hidden by the subscription filter. Updates automatically every time the dashboard redraws.

## Result

Version 2.2.38 makes it possible to tell at a glance whether the whitelist is fully represented on the dashboard or something is missing.
