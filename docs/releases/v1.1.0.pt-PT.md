# Twitch Drop Farmer v1.1.0

Esta versão foca-se na fiabilidade, numa interface mais fluída e numa melhoria clara do carregamento das imagens dos jogos.

## Destaques

- Carregamento assíncrono das capas dos jogos para manter o dashboard e a vista de farming responsivos.
- Nova cadeia de resolução de imagens com prioridade para Twitch, depois Steam e por fim Google Images.
- Placeholder visual gerado com as iniciais do jogo quando não existe nenhuma imagem real disponível.
- Novo botão de atualização manual do dashboard para forçar refresh dos dados e das imagens.

## Correções

- Redução de bloqueios longos nas chamadas GraphQL com um limite total de retry.
- Parsing de timestamps endurecido contra dados inválidos devolvidos pela Twitch.
- Limpeza correta do fallback via browser para evitar fugas de recursos Qt.
- Melhor deteção de campanhas concluídas que continuam marcadas como ativas após terminar as recompensas.
- Resultados falhados de procura de imagens deixaram de ficar em cache permanente, permitindo novo retry mais tarde.
- Verificação prévia das URLs de artwork da Steam antes de as usar.

## Resultado

O programa deverá agora sentir-se mais rápido durante os refreshes, recuperar melhor de falhas temporárias da Twitch e apresentar muito menos jogos sem imagem.
