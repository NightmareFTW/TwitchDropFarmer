# Twitch Drop Farmer v2.2.41

This release fixes the Filters whitelist showing an incomplete game list and lets you add any game by hand.

## Why

The full list of games with active Twitch Drops comes from Twitch's `ViewerDropsDashboard` query, which Twitch intermittently blocks with an "integrity check". During those periods only the in-progress Inventory games are obtainable — by any method, even a real browser. So a user with dozens of whitelisted games would only see the ones currently being farmed; the rest disappeared from the list and couldn't even be unchecked.

## Fixed

- The Filters game list is now the **union** of the current catalog (snapshot or disk cache), the already-configured whitelist/blacklist, and any manually-added games. Every configured game is always visible and manageable, regardless of whether it has an active campaign at that moment.

## Added

- **Manual "add game"** field + button in the Filters tab. Type a game's exact name to whitelist it even when Twitch is withholding the full catalog; it becomes selected and is saved on Save settings.

## Result

Version 2.2.41 means your whitelist always shows all your games, and you can always add new ones — you're no longer limited to whatever Twitch happens to expose that session.
